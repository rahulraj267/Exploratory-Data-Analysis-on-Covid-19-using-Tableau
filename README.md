Exploratory Data Analysis on Covid - 19 
• Conducted a comprehensive exploratory data analysis on Covid-19 data using Python and Tableau to gain insights into the pandemic's impact. Employed data cleaning and pre-processing techniques with Pandas and NumPy.
• Utilized visualization libraries such as Matplotlib and Seaborn to create various types of plots, including line charts and bar charts, to visually represent the data and uncover patterns and trends. 
• Analyzed the data to draw insights on the regional impact of the pandemic and mortality rates to gain a better understanding of the pandemic's impact. 
• Developed a dynamic dashboard using Tableau to visualize the analysis results and provide a clear representation of the data insights.
